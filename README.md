# expogo2026
## following the instructions in March 2026

TLDR;

Install Expo Go on your device.

In the codespace created from this template run

```bash
npm install
npx expo start --tunnel
```

Scan the qrcode from the terminal window, and you should be able to see your app.

Use the import in index.js to integrate your /src folder.
Remove the App.js

You should be able to see your loyalty/ordering app on your phone
